Proyecto

Aplicación Web para el intercambio de cartas Magic: The Gathering, en la cual, cualquier usuario previamente registrado podrá subir sus cartas intercambiables y buscar cartas de otros usuarios con la posibilidad de proponerles un intercambio. Cada usuario podrá consultar los intercambios que le han propuesto, para aceptarlos, rechazarlos o realizar una contraoferta al usuario que le ha propuesto el cambio. Tendra una semana para actuar, si no la oferta expirará. En caso de aceptarlos, la aplicación compartirá los datos de contacto entre los usuarios involucrados en el cambio, para que estos realicen físicamente el cambio de la forma que consideren (presencial, envío por correo, etc). Despues de aceptar el intercambio, ambos deben confirmar que ha tenido lugar. Una vez finalizado un intercambio no se podra modificar. Los usuarios podrán valorar el trato del otro usuario solo una vez. La información sobre la valoración de cada usuario podrá ser consultada antes de realizar un intercambio. Los usuarios podrán ser coleccionistas, con todas las funcionalidades anteriormente citadas, o administradores, que podrán eliminar usuarios o actualizar la base de datos de cartas.
